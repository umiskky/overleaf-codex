# Code of Conduct

This project should be practical, respectful, and welcoming to contributors with
different levels of experience.

## Expected behavior

- Be direct and constructive.
- Explain technical tradeoffs with evidence.
- Keep discussions focused on the project and user safety.
- Respect maintainers' time by providing clear reproduction steps and focused
  pull requests.

## Unacceptable behavior

- Harassment, discrimination, or personal attacks.
- Publishing private credentials, paper content, or project data.
- Repeatedly derailing technical discussions.
- Misrepresenting `olcx` as an official Overleaf project.

## Enforcement

Maintainers may edit or remove comments, close issues, reject contributions, or
block accounts when behavior harms the project or its users.

Report concerns through the repository security/contact channels. Do not include
secret tokens, cookies, or private paper material in public issues.
